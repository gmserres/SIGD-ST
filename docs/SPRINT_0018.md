# Sprint 0018 - Comparador OP y Facturas

## Objetivo

Usar una OP completa como caso patrón para avanzar desde extracción simple hacia comparación documental.

## Caso patrón

OP MONKE WALTER ANDRES:

- Proveedor: MONKE WALTER ANDRES.
- CUIT: 20-16898364-7.
- Monto total de facturas: $840.000,00.
- Retenciones: Ganancias, Ingresos Brutos e IVA.
- Monto neto a pagar: $706.140,03.
- Facturas liquidadas: 6.

## Implementado

- Extracción de facturas liquidadas.
- Extracción de retenciones.
- Separación entre importe bruto y neto.
- Comparación de suma de facturas contra monto total.

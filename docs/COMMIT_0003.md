# Commit 0003

## Objetivo

Convertir el frontend inicial en una primera interfaz operativa del SIGD-ST orientada al expediente.

## Implementado

- Panel principal.
- Menú lateral.
- Nuevo expediente.
- Mis expedientes.
- Ficha del expediente.
- Documentos asociados.
- Historial del expediente.
- Acciones de validación.
- Acción para generar disposición.

## Flujo habilitado

1. Crear expediente.
2. Ver expediente en listado.
3. Abrir ficha.
4. Validar expediente.
5. Generar disposición.
6. Consultar historial.
